# Power-up upgrade icons

Placeholder art for the power-up chooser (`scripts/powerup_select.gd`), one PNG per upgrade
**type**. Looked up by convention via `scripts/data/upgrade_icons.gd`.

These are **optional placeholders** (generated by `tools/gen_upgrade_icons.gd`): the chooser falls
back to a flat colour swatch when a file is missing, so the game runs without them. To replace one
with real art, drop a same-named PNG in here and re-run `--import`.

## Spec

| | |
|---|---|
| **Filename** | `<type>.png` — one of `hp`, `attack`, `defense`, `move` |
| **Size** | 128 × 128 (square); other square sizes scale fine |
| **Format** | PNG; opaque or transparent both work |
| **Shown** | On the upgrade cards in the power-up chooser (the 3 choices offered per power-up node) |

## Regenerate the placeholders

```bash
"$GODOT" --headless --path "C:\\Users\\Dad\\GriffinsGame" --script res://tools/gen_upgrade_icons.gd
"$GODOT" --headless --path "C:\\Users\\Dad\\GriffinsGame" --import
```
